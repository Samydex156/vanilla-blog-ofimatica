Atajos de Teclado Esenciales para Windows 10

Domina tu computadora sin tocar el ratón. Esta guía reúne los atajos más útiles para el día a día.

Generales del Sistema

Atajo Acción
Ctrl + C Copiar elemento seleccionado
Ctrl + X Cortar elemento seleccionado
Ctrl + V Pegar elemento copiado o cortado
Ctrl + Z Deshacer última acción
Ctrl + Y Rehacer acción deshecha
Ctrl + A Seleccionar todo
Ctrl + S Guardar archivo actual
Ctrl + B Cerrar ventana o pestaña activa
Alt + F4 Cerrar ventana o apagar el equipo
F2 Renombrar elemento seleccionado
F5 Actualizar ventana o recargar página
Supr Enviar a la Papelera de reciclaje
Shift + Supr Eliminar permanentemente (no va a la Papelera)

Tecla Windows (Win)

Atajo Acción
Win Abrir Menú Inicio
Win + D Mostrar u ocultar el escritorio
Win + E Abrir el Explorador de archivos
Win + I Abrir Configuración de Windows
Win + L Bloquear el equipo (ideal al levantarte)
Win + R Abrir el cuadro "Ejecutar"
Win + S Abrir búsqueda de Windows
Win + V Abrir historial del portapapeles
Win + . Abrir panel de emojis
Win + Tab Vista de tareas (escritorios virtuales)
Win + Flecha Izq/Der Acoplar ventana a un lado de la pantalla
Win + M Minimizar todas las ventanas

Explorador de Archivos

Atajo Acción
Ctrl + Shift + N Crear una carpeta nueva
Ctrl + Shift + E Mostrar todas las carpetas del panel izquierdo expandidas
Alt + Flecha Arriba Ir a la carpeta padre (subir un nivel)
Alt + Flecha Izq Ir a la carpeta anterior (atrás)
Alt + Flecha Der Ir a la carpeta siguiente (adelante)
F11 Pantalla completa en el Explorador
Tip del Profesor El atajo que más tiempo te ahorrará es Win + E para abrir el Explorador de archivos al instante. Combínalo con Ctrl + Shift + N para crear carpetas sin tocar el mouse.
