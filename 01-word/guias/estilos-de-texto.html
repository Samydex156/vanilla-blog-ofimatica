ESTILOS DE TEXTO: NEGRITA, CURSIVA Y SUBRAYADO

En esta guía aprenderás a usar los estilos básicos de fuente para destacar elementos importantes en tus documentos de Word.

Los estilos de texto te permiten dar énfasis a ciertas partes de tu documento. Para aplicar cualquiera de estos estilos, los pasos a seguir son siempre los mismos:

  1. Selecciona el texto que deseas modificar (una letra, palabra, frase o párrafo entero).
  2. Ve a la pestaña INICIO en la cinta de opciones.
  3. Haz clic en el botón correspondiente en el grupo Fuente, o utiliza los atajos de teclado.

Tipos de Estilos Básicos

  • Negrita: Hace que el texto sea más grueso y oscuro. Se usa comúnmente para títulos, subtítulos o para resaltar las palabras clave de un párrafo.
    Atajo de teclado: Ctrl + N

  • Cursiva: Inclina ligeramente el texto hacia la derecha. Se utiliza frecuentemente para títulos de obras (libros, películas), palabras en otros idiomas o para dar un énfasis sutil.
    Atajo de teclado: Ctrl + K

  • Subrayado: Traza una línea por debajo de las palabras. Al igual que la negrita, ayuda a destacar texto importante. Si haces clic en la flecha al lado del botón, puedes elegir diferentes estilos de línea.
    Atajo de teclado: Ctrl + S

Quitar un formato

Si deseas eliminar la Negrita, Cursiva o Subrayado que aplicaste por error, simplemente vuelve a seleccionar el texto y haz clic de nuevo en el mismo botón, o presiona el mismo atajo de teclado para desactivarlo.
