Guía Atajos de Teclado

Para Microsoft Word.

Atajos de teclado

Un atajo de teclado es una combinación de teclas que se presionan para realizar una acción específica.

Atajo Acción Referencia
Ctrl + G Guardar los cambios realizados de un documento Word.
F12 Abre la ventana de "Guardar como" para guardar una copia del documento.
Ctrl + U Crea un nuevo documento de Word
Ctrl + N Aplica estilo de texto Negrita al texto seleccionado.
Ctrl + S Aplica estilo de texto Subrayado al texto seleccionado.
Ctrl + K Aplica estilo de texto Cursiva al texto seleccionado.
Ctrl + C Copia el texto seleccionado.
Ctrl + V Pega el texto seleccionado en la posición del cursor.
Shift + F3 Cambia a Mayúsculas / Minúsculas / Tipo oración al texto seleccionado.
Ctrl + E Selecciona todo el contenido del documento.
Shift + Inicio Selecciona toda la línea de texto hacia la izquierda desde la posición del cursor.
Shift + Fin Selecciona toda la línea de texto hacia la derecha desde la posición del cursor.
Ctrl + Scroll Efecto de Zoom (Acerca o alejar la hoja del documento)
