Atajos de Teclado: Excel Esencial

Aumenta tu productividad en el instituto Nueva Tecnología

1. Gestión de Archivos y Libros

Acción Combinación de Teclas
Crear un nuevo libro Ctrl + U
Guardar el libro actual Ctrl + G
Imprimir documento Ctrl + P

2. Edición y Formato

Acción Combinación de Teclas
Copiar selección Ctrl + C
Pegar selección Ctrl + V
Deshacer última acción Ctrl + Z
Aplicar Negrita Ctrl + N
Tip Pro: Usa F2 para entrar directamente a editar el contenido de una celda sin tener que usar el mouse.

3. Navegación Rápida

Moverse por grandes tablas de datos en la UMSA o en el trabajo es más fácil así:

  • Ir al final de un rango de datos: Ctrl + Flechas
  • Seleccionar hasta el final de los datos: Ctrl + Shift + Flechas
