Contenido del Libro de Trabajo

Este libro de Excel, como su nombre lo indica, es una miscelánea de ejercicios diseñados para practicar múltiples funciones y herramientas vistas en clase. Dentro de este archivo encontrarás varias hojas de cálculo con retos específicos que te exigirán aplicar lo siguiente:

  • Ordenar y Filtrar Registros: Ejercicios para manejar listas de datos, ordenarlos mediante diferentes criterios y utilizar filtros para encontrar información específica de manera rápida.
  • Fórmulas y Funciones Básicas: Retos que incluyen la aplicación de funciones estadísticas clave, como la función PROMEDIO para el análisis rápido de datos numéricos.
  • Función Lógica SI: Problemas que requieren evaluar condiciones lógicas para devolver resultados distintos según se cumplan o no ciertos escenarios.
  • Tablas Dinámicas: Prácticas esenciales para aprender a resumir, analizar, explorar y presentar datos estructurados de forma interactiva y profesional.
  • Herramientas Útiles Variadas: Diferentes tareas que pondrán a prueba el uso de otras herramientas importantes de Excel que facilitan el trabajo y la limpieza de datos.

Asegúrate de revisar minuciosamente las instrucciones dentro de cada hoja del libro descargable para completar los ejercicios con éxito.
