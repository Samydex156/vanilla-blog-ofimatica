Conceptos esenciales para maquetación en el instituto Nueva Tecnología

1. Elementos Fundamentales

Elemento Descripción
Cuadros de Texto Contenedores independientes para escribir. Se pueden enlazar para que el texto fluya de uno a otro.
Marcadores de posición Espacios reservados para imágenes que permiten diseñar la estructura antes de tener el contenido final.
Guías de Diseño Líneas no imprimibles que ayudan a alinear objetos (Rosadas para márgenes, Azules para objetos).

2. Tipos de Publicaciones Comunes

  • Trípticos: Folletos doblados en tres partes, ideales para información turística o institucional.
  • Tarjetas de Presentación: Documentos pequeños con datos de contacto.
  • Boletines: Publicaciones periódicas con noticias y artículos.
Tip de Diseño: Mantén siempre un margen de seguridad ("zona de sangría") si vas a imprimir tu trabajo en una imprenta profesional, para evitar que se corten textos o imágenes.

3. Atajos Útiles de Teclado

Agiliza tu flujo de trabajo de diseño:

Atajo de teclado Descripción / Acción
Ctrl + G Agrupar objetos
Ctrl + Shift + G Desagrupar objetos
Alt + F6 Traer al frente
Alt + Shift + F6 Enviar al fondo
