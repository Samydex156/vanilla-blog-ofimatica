# Recursos Ofimática — Blog Educativo

Portal web de recursos educativos para el curso de **Ofimática** del **Instituto Nueva Tecnología**, impartido por el **Prof. Samuel Durán**. Clases de teoría, guías prácticas y ejercicios descargables sobre herramientas ofimáticas esenciales.

## Características principales

-   **7 módulos completos**: Windows, Word, PowerPoint, Excel, Publisher, Internet e Inteligencia Artificial
-   **Clases de teoría** con contenido visual y explicaciones paso a paso
-   **Guías prácticas** con atajos de teclado, tips y procedimientos detallados
-   **Ejercicios descargables**: archivos .docx, .xlsx y .pdf para practicar
-   **Buscador en vivo** unificado para filtrar contenido al instante
-   **Modal de imágenes** con zoom para ver capturas de pantalla a detalle
-   **Diseño responsive**: óptimo en desktop, tablet y móvil
-   **Modo oscuro** automático (sigue la preferencia del sistema) y manual (con toggle)
-   **Vista de impresión** optimizada: encabezado con datos del profesor, pie con número de página, tamaño carta
-   **100% estático**: sin frameworks, sin base de datos, despliegue instantáneo

## Atajos de navegación

| Atajo | Acción |
|-------|--------|
| Clic en ícono de imagen | Abre modal con zoom completo |
| Clic en fondo del modal | Cierra la imagen ampliada |
| Buscador en índices | Filtra en tiempo real la lista de contenidos |
| Botón ☀/☾ en header | Cambia entre modo claro y oscuro |

## Requisitos previos

Navegador web moderno actualizado (Chrome, Firefox, Edge, Safari).

## Instalación y configuración rápida

```bash
# 1. Clonar el repositorio
git clone https://github.com/samuelduran/vanilla-blog-ofimatica.git
cd vanilla-blog-ofimatica

# 2. No requiere instalación de dependencias
#    El proyecto es HTML/CSS/JS puro, sin build system.

# 3. Abrir localmente
start index.html

# 4. (Opcional) Despliegue en Vercel
#    Conectar el repositorio a Vercel — sin configuración adicional.
```

## Despliegue

El sitio se despliega automáticamente en **Vercel** con la configuración en `vercel.json`:
- URLs limpias (sin extensión `.html`)
- Sin trailing slashes
