CÓMO USAR ZOOMIT PARA TUS PRESENTACIONES

Aprende a hacer zoom, dibujar en la pantalla y poner temporizadores en vivo durante tu exposición.

ZoomIt (de Sysinternals/Microsoft) es una pequeña utilidad gratuita indispensable para cualquier persona que da presentaciones, clases o tutoriales técnicos. Se ejecuta en segundo plano y te permite interactuar con tu pantalla (incluso fuera de PowerPoint) con simples atajos de teclado.

Cómo Descargar e Instalar

ZoomIt no requiere una instalación compleja, es un programa "portable". Sigue estos pasos:

  1. Ve a la página oficial de Microsoft para descargar la herramienta: Descargar ZoomIt (Sysinternals).
  2. Haz clic en el enlace de descarga (generalmente dice "Download ZoomIt").
  3. Se descargará un archivo comprimido .zip. Haz clic derecho sobre él y selecciona "Extraer todo...".
  4. Abre la carpeta extraída y haz doble clic en el archivo ZoomIt.exe.
  5. Aparecerá la ventana de configuración. Te aconsejamos marcar la casilla "Run ZoomIt when Windows starts" abajo a la izquierda para que siempre esté listo para usarse. Luego dale a OK.

1. Función de Zoom Básico y Livezoom

A veces los asistentes en la parte de atrás no alcanzan a ver un botón o número pequeño. ZoomIt tiene dos modos de acercamiento:

  • Zoom Convencional (Ctrl + 1): Congela la pantalla y hace un acercamiento. Puedes moverte con el ratón o usar la rueda para acercar más. Para salir, presiona Esc o clic derecho.
  • LiveZoom (Ctrl + 2): Hace un acercamiento pero sin congelar la pantalla. ¡Puedes seguir haciendo clics, escribiendo y usando programas mientras estás en zoom!

2. Modo Dibujo y Formas

Puedes empezar a dibujar sobre cualquier pantalla en cualquier momento presionando Ctrl + 3. O si ya estás en el "Zoom Convencional", simplemente haz clic izquierdo para empezar a dibujar.

Formas Geométricas (Mantén presionada la tecla dibujando)
  • Shift + Arrastrar: Línea recta.
  • Ctrl + Arrastrar: Rectángulo.
  • Tab + Arrastrar: Círculo / Elipse.
  • Shift + Ctrl + Arrastrar: Flecha (ideal para señalar).

Ajustar el Grosor del Lápiz

Mientras estás en modo dibujo, puedes cambiar el grosor de la línea de dos maneras:

  1. Mantén presionado Ctrl y usa la Rueda del ratón hacia arriba o hacia abajo.
  2. Mantén presionado Ctrl y pulsa las teclas de Flecha Arriba o Flecha Abajo.

3. Cambiar Colores del Marcador

Mientras estás dibujando, simplemente presiona una tecla para cambiar el color de la tinta al instante (se basan en las iniciales de los colores en inglés):

Tecla Color
RRojo (Red)
GVerde (Green)
BAzul (Blue)
YAmarillo (Yellow)
ONaranja (Orange)
PRosa / Púrpura (Pink/Purple)

4. Escribir Texto en Pantalla

Si dibujar letras con el ratón es muy difícil, puedes escribir con el teclado:

  1. Entra al modo dibujo (Ctrl + 3).
  2. Presiona la tecla T.
  3. Haz clic donde quieras escribir y tipea tu texto. (Puedes cambiar el color antes presionando las teclas de colores, o usar la rueda del ratón para agrandar/achicar la fuente).
  4. Haz clic en otro lado para salir del modo texto, o Esc para borrar todo.

5. Copiar o Guardar la Pantalla

Si has hecho anotaciones importantes, zoom o dibujos en la pantalla y quieres guardarlos para compartirlos con tus alumnos:

  • Copiar (Ctrl + C): Estando en el modo de Zoom o de Dibujo, presiona esta combinación para copiar todo lo que ves al portapapeles. Luego puedes pegarlo (Ctrl + V) en Word, PowerPoint o WhatsApp.
  • Guardar (Ctrl + S): Te abrirá una ventana para guardar la captura de pantalla dibujada directamente como una imagen en tu computadora.

6. Temporizador de Descanso (Break Timer)

¿Vas a dar un receso de 10 minutos a tus estudiantes o público? No mires el reloj de pared. Presiona Ctrl + 4.

La pantalla se oscurecerá o mostrará un fondo blanco y aparecerá un reloj de cuenta regresiva gigante. Puedes aumentar o disminuir los minutos usando la rueda del ratón o las flechas Arriba/Abajo. ¡Muy profesional!

Pizarras Rápidas

Si estás en modo dibujo (Ctrl + 3), pero la pantalla actual tiene mucho ruido visual, presiona W para volver toda la pantalla blanca (Whiteboard), o K para volverla completamente negra (Blackboard). Ideal para explicar un concepto desde cero con tus marcadores de colores.
